搞钱 Wiki 收录两类结构化资料：**信用卡**与**券商**。每一条目都带有统一的字段（年费、佣金、权益等），因此可以在目录页中按条件筛选比较。

除了资料库，本站还有两组说明文章：

- **信用卡**：如何选卡、如何用卡。
- **投资**：投资基础概念与风险认知。

:::caution[免责声明]
本站内容仅供学习与参考，不构成任何投资建议或金融产品推荐。文中提及的银行、券商及产品均为虚构示例。
请在做出任何财务决策前咨询持牌专业人士，并自行核实产品条款。
:::
