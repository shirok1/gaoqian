债务失控时，最重要的不是寻找一笔更大的新贷款，而是停止增加复杂度。先保证基本生活，再把本金、综合成本、到期日和担保情况放到同一张表里。

## 第一步：做完整债务清单

每笔至少记录：

| 字段 | 为什么重要 |
| --- | --- |
| 剩余本金 | 判断真实规模 |
| 年化利率及所有费用 | 比较资金成本 |
| 最低还款额、到期日 | 避免遗漏 |
| 是否逾期、逾期后果 | 判断紧急程度 |
| 是否有抵押、保证人 | 识别资产和他人风险 |
| 官方客服与合同编号 | 后续核实、协商 |

贷款营销依法应明示年化利率，且成本应包含与贷款直接相关的各类费用。只看到日息或月费率时，回到合同和还款计划核对综合年化成本。

## 第二步：先保护底线

发薪后先留出住房、基本饮食、必要医疗、通勤和维持工作的费用。把全部现金交给某一家债权人、随后又靠新借款生活，通常只会重新开始循环。

然后处理会立即危及基本住房、必要生产工具或保证人的债务，再确保各笔债务不因操作疏忽产生额外违约。具体法律后果因合同而异，不确定时咨询正规法律援助。

## 第三步：选择一种排序并坚持

- **高息优先法**：其他债务按约定支付，把额外资金集中到综合成本最高的一笔，通常总成本更低。
- **小额优先法**：先清偿余额最小的一笔，减少账户数量，适合需要阶段性反馈的人。

两种方法都比随意还款更容易执行。涉及已逾期、抵押或诉讼风险时，不应只按利率机械排序。

## 现金流已经不够怎么办

尽早通过贷款机构官方客服说明情况，询问是否有合规的展期、分期或困难支持方案，并要求书面确认费用和征信报送安排。不要向陌生“债务优化”“征信修复”机构支付前期费用，不伪造证明，也不失联。

保留合同、账单、还款凭证和沟通记录。遇到威胁、暴力催收或涉嫌诈骗时，保存证据并向公安机关或相应监管投诉渠道反映。

清偿完成后索取结清证明，继续保留一段时间，并通过中国人民银行征信中心等官方渠道核对信用报告。

:::caution
本文是通用整理框架，不构成法律、信贷或债务重组建议。已经逾期、被起诉、存在抵押物处置风险或无力保障基本生活时，应尽快联系债权人并寻求当地法律援助等正规支持。
:::
