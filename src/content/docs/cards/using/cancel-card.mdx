办卡有无数篇攻略，销卡却几乎没人认真讲——于是每年都有人栽在同一个坑里：卡剪了，账户还活着，第二年一笔年费悄悄入账，逾期几个月后才在征信报告上和它重逢。这篇把销卡这件"小事"的完整流程讲清楚：销卡和销户的区别、动手前的清算清单、年费卡的时点选择，以及怎么防住"幽灵年费"。

## 销卡 ≠ 销户

先分清两个概念，这是整篇文章最重要的一对名词：

- **销卡**：注销某一张卡片。卡不能刷了，但你在这家银行的**信用卡账户可能还在**——尤其当你在同一银行有多张卡时，它们通常共享一个账户。
- **销户**：注销你在这家银行的整个信用卡账户，名下该行所有信用卡一并失效，与银行的信用卡合约关系彻底终结。

问题出在"只销卡不销户"的状态：账户还活着，就仍然可能产生费用（部分卡种的年费挂在账户或卡产品上）、仍然占用你在这家银行的授信记录。**如果你的目标是和这家银行的信用卡业务彻底了断，要的是销户，不是销卡。** 打客服电话时明确说"销户"，并问清楚名下是否还有其他卡片或附属卡挂在同一账户下。

## 销卡前的清算清单

动手之前，把账面清零。四件事按顺序做：

1. **清积分。** 销卡销户后积分通常直接作废，部分银行同一账户下可保留，但别赌——把能兑的兑掉。参考[积分与返现](/cards/using/points/)里的估值思路，兑不出像样价值的就当放弃，别为了几千积分留着一张要收年费的卡。
2. **清欠款。** 包括还没出账单的消费、分期的剩余期数（注意：**未到期的分期通常需要一次性结清，且已收的手续费不退、未收的可能照收**，规则以银行为准）。
3. **清余额（溢缴款）。** 还多了的钱、退款回来的钱趴在卡里，这叫溢缴款。销卡前转出或消费掉——销户后再取会很麻烦。
4. **解绑自动扣款。** 视频会员、云存储、打车软件的免密支付……逐个换绑到其他卡。漏掉一个，轻则扣款失败服务中断，重则在你以为已经销掉的卡上产生新交易，导致销户失败。

清单走完，再打客服电话提销户。

## 年费卡的销卡时点

有刚性年费或高门槛年费的卡，**销卡时点直接决定你是否多付一年年费**。原则很简单：**赶在下一个年费计费日之前完成销户。**

难点在于弄清自己的年费什么时候入账——有的按核卡日起算的"卡年度"收，有的按自然年收，各行不同。打客服问两个问题："我的年费下次入账是哪天？""现在销户，这笔年费还收吗？"

两个常见情况：

- **年费刚入账不久**：部分银行支持在入账后短期内销户并退还或减免当期年费，值得开口争取——客服挽留话术后面往往跟着减免权限。
- **刷免条件差一点点**：如果差一两笔交易就能免年费，先刷够再销，别硬亏一笔年费（刷免规则详见[年费与免年费](/cards/choosing/annual-fee/)）。

:::tip
销户电话里，客服大概率会挽留：送积分、免年费、升级权益。别不好意思听完——**挽留礼有时比你销卡的理由更值钱**，尤其是高年费卡的年费减免。听完再决定：条件真香就留一年，不香就坚持销户，主动权在你。
:::

## 销卡对征信的影响

先安抚一下：**正常状态下销卡销户，不是征信污点。** 账户会以"销户"状态体现在报告里，已有的良好还款记录仍然保留，构成你信用历史的一部分。

但有三件事要想清楚：

- **销卡不能消除逾期记录。** 有人以为把有逾期的卡销掉就"翻篇"了——恰恰相反。不良记录自**结清之日**起保留 5 年，销户只是让这个账户停止产生新记录。圈内的常见建议是：出过逾期的卡，结清后**继续正常使用一段时间**，用新的按时还款记录覆盖旧的负面记录，比立刻销户更有利（详见[征信：你的金融档案](/cards/using/credit-report/)）。
- **最老的账户有历史价值。** 账户年龄是信用评估的维度之一，你的第一张卡承载着最长的记录。如果它没有年费负担，留着当"资历卡"比销掉好。
- **销户释放授信额度。** 反过来，如果你总授信过高影响了房贷审批，销掉闲置卡反而是加分操作。

## 防"幽灵年费"复活

最后是本文存在的核心理由。"幽灵年费"的剧本是这样的：你以为销卡成功了，但账户没销干净，或销户流程未走完；次年年费照常入账；卡已剪、App 已删、短信提醒早就退订，你对这笔欠费一无所知；几个月后它变成征信上的连续逾期。防御动作四条：

1. **销户时问清生效时间。** 多数银行销户有 30–45 天左右的清算期（防止退款、延迟交易入账），清算期结束才真正销户。
2. **清算期结束后回访确认。** 打客服核实"账户已注销、余额为零、无任何欠费"，最好让客服发送确认短信。
3. **销户后 1–2 个月再查一次。** 登录一次网银或打一次电话，确认没有新产生的费用。
4. **半年后核对征信。** 免费查一次征信报告，确认该账户状态显示为"销户"、无逾期。

:::caution
**"卡剪了"和"账户销了"之间的距离，是每年无数条征信逾期记录的来源。** 剪卡只是物理动作，对账户状态没有任何影响。销户以银行系统里的账户注销为准、以你的回访确认为准——在拿到"账户已注销、无欠费"的明确答复之前，这件事没有结束。
:::

销卡的完整姿势总结成一句话：**清空账面、掐准时点、明确销户、事后确认。** 四步都做到，才算真正和一张卡好聚好散。
