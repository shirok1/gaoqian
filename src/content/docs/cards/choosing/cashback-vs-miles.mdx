在[积分与返现](/cards/using/points/)里我们讲过方法论：把积分按你实际会用的兑换渠道折算成现金，和返现放在同一个维度比较。这一篇是那套方法论落到选卡上的应用——当你站在"办一张返现卡还是一张里程卡"的岔路口时，怎么用自己的消费结构和出行频率做决策。**答案不在卡的参数表里，在你自己的账单和日历里。**

## 决策只看两个变量

选卡文章总喜欢罗列十几个对比维度，其实对返现 vs 里程这个问题，砍到最后只剩两个变量：

**变量一：年飞行次数。** 里程的价值出口只有一个——机票（以及少量升舱）。你一年飞几次、飞什么航线、买什么舱位，直接决定里程在你手里能折出多少钱。不飞的人，里程价值无限趋近于零，这和卡的积分规则多慷慨没有任何关系。顺带检查一句：住店频率高的人，酒店积分体系是另一条平行的路，逻辑与里程完全同构。

**变量二：年消费体量。** 积分玩法的收益与刷卡量正相关，而且不是线性的——里程卡的年费、多倍积分活动的达标门槛、里程兑换的起兑数量，都是固定成本。**消费体量不够，这些固定成本摊不薄，再高的名义回馈率也是纸面数字。** 粗略地说，如果你的月刷卡额只有几千元，一年攒下的里程可能连一张像样的国内单程都换不出来。

## 四象限：把自己放进去

两个变量交叉，得到四个象限。对号入座：

**高飞行 + 高消费：里程卡的目标客群。** 一年飞六次以上、月消费体量上万，你就是里程体系设计时瞄准的那类人。里程卡的高年费在你这里能被兑换价值覆盖，多倍活动你能达标，兑换的机票是你本来就要买的。这个象限里，认真研究里程卡是值得的——记得用"元/千分"把候选卡折算成真实回馈率再比。

**高飞行 + 低消费：先算再选，通常先返现。** 你经常飞，但刷卡量撑不起里程的积累速度——攒两年才够一张短途兑换，期间还要提防积分贬值。这个象限的常见解法：**主力用免年费的返现卡，飞行价值靠航司常旅客计划本身积累**（坐飞机送的里程和刷卡无关），等消费体量上来了再考虑里程卡。

**低飞行 + 高消费：返现卡，别犹豫。** 这是最容易选错的象限——消费能力强的人容易被高端里程卡的权益包装吸引。但你不飞，攒下的巨量里程只能贬值或过期，兑换出口是虚的。高消费在返现卡上能拿到确定的真金白银，**确定的 1% 胜过用不掉的"名义 3%"。**

**低飞行 + 低消费：免年费返现卡，句号。** 学生、刚工作的新人多数在这一格。这个阶段的任务是建立信用记录和还款习惯（见[如何挑选第一张卡](/cards/choosing/first-card/)），回馈是顺带的。任何需要年费投入的卡在这个象限都不划算。

:::tip
判断自己的位置时，用**过去十二个月的真实数据**：翻航旅 App 数一数实际飞了几次，翻账单看月均刷卡额——而不是"我今年应该会多飞几次"的展望。**用愿望选卡是里程卡玩家亏钱的第一大原因**，飞行计划落空，年费照收不误。
:::

## 别为用不掉的权益付年费

里程卡的定价结构里，年费买的是一个"权益+积分"的打包组合：贵宾厅、接送机、多倍里程累积。评估它值不值，方法在[年费与免年费](/cards/choosing/annual-fee/)里讲过——**只把你确定会用到的权益按市价折算，加总后和年费比**。这里补充里程卡特有的三个坑：

- **权益按次数计价，但你的使用次数有上限。** 十次贵宾厅听着丰厚，你一年飞四次，就只值四次的钱（贵宾厅本身值不值另说，见[休息室](/cards/choosing/lounge/)的分析）。
- **里程有隐性折损。** 兑换要看放票、要付税费和可能的燃油附加费、积分会贬值——给里程估值时打个七八折才接近现实。
- **达标门槛是隐藏成本。** "消费满 N 万享次年免年费/送里程"的门槛，如果需要你刻意凑消费才够得着，凑出来的消费本身就是亏损。

一句检验话术：**如果这张卡明天把权益全部砍掉、只剩积分规则，你还愿意付这个年费吗？** 如果答案是否定的，再逐项审视那些权益你到底会不会用。

:::caution
警惕"权益幻觉"：宣传页上的机场接送、高尔夫、酒店升房，会让一张年费上千的卡显得"办了就赚"。但权益的纸面价值和你的实际使用价值是两回事——**没用掉的权益价值为零，而年费是真金白银地扣。** 近几年银行权益整体在缩水，办卡时看到的权益清单，第二年未必还在。以银行官方最新公告为准，且给权益缩水留出心理余量。
:::

## 决策之后：动态复查

这个决策不是一锤子买卖。生活状态一变——换了要频繁出差的工作、从月光变成有稳定结余、或者反过来——你的象限就变了。**每年年费入账前复查一次**：过去一年实际飞了几次？里程兑出去了吗？折算的回馈率还打得过免年费返现卡吗？答案变了就换卡，用[销卡的正确姿势](/cards/using/cancel-card/)体面退场。

最后回到那句老结论：返现和里程没有优劣，只有匹配。**里程卡是给"本来就要飞"的人放大出行价值的工具，不是让不飞的人开始飞的理由。** 选卡之前先认清自己是谁，这比研究任何一张卡的参数都重要。
